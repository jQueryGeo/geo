shapeStyle

return type Object ( geomap style ) | jQuery
syntax .geomap( "shapeStyle", Object style )
usage 
var currentStyle = $("#map").geomap( "shapeStyle" )
$("#map").geomap( "shapeStyle", { width: "16px", height: "16px" } )

The shapeStyle method retrieves or updates the default style of geomap graphics. The reasons that this is a method instead of a property will be clearer in a future release.

This method changes specific properties of the internal default style. If you send an incomplete style object, only the style properties you reference are updated.

You can override this style with every call to addShape.

Changing this default style will update the style of any existing shape that has been added with addShape but not given its own style. For example, if you add two GeoJSON Point objects and in the first addShape call you do not pass a style but in the second one you do. In this situation, future calls to shapeStyle will only update how geomap draws the first shape.

When called with a second argument to set the shapeStyle, this method returns the original jQuery collection to maintain call chaining.

geomap style

A geomap style is an object whose properties follow a subset of SVG styling properties. The specific styles that geomap recognizes and to which geometry they apply are listed below.

property default description
color black An indirect, fallback value for the fill and stroke properties if they are not set.
fill undefined Draws the interior of a shape. The area to be drawn consists of any areas inside the outline of the shape. By default, fill will use the value of the color property.
fillOpacity undefined Specifies the opacity of the drawing operation used to draw the interior of a shape. By default, fillOpacity will use the value of the opacity property.
height "8px" The height of an oval drawn around Point shapes. Currently only pixel values are allowed.
opacity 1 An indirect, fallback value for the fillOpacity and strokeOpacity properties if they are not set.
stroke undefined Draws along the outline of a shape. By default, stroke will use the value of the color property.
strokeOpacity undefined Specifies the opacity of the drawing operation used to draw the outline of a shape. By default, strokeOpacity will use the value of the opacity property.
strokeWidth "1px" The width of the stroke of a shape. A zero value causes no stroke to be drawn.
visibility "visible" Determines if the shape is drawn ("visible") or not drawn ("hidden") on the map. Shapes that are hidden can still be returned by the findShape method.
width "8px" The width of an oval drawn around Point shapes. Currently only pixel values are allowed.

All properties apply to Point shapes which means that you can adjust the stroke and fill of the oval surrounding the point location.
