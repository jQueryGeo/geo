toMap

return type Array | Array^2 (GeoJSON position)
syntax .geomap( "toMap", Array pixelPosition | Array^2 pixelPositions )
usage var mapPos = $("map").geomap( "toMap", [ 120, 240 ] )
var mapPositions = $("map").geomap( "toMap", [ [ 120, 240 ], [ 192, 48 ] ] )

The toMap method takes a single pixel position (which is an array of two values: the x and y of the pixel) or an array of pixel positions and returns the map coordinates for the pixels.

If $.geo.proj is not null, the widget returns the positions as geodetic coordinates. Otherwise, the widget returns positions in its internal projection as defined by tilingScheme for tiled map services or bbox for shingled (dynamic) map services.
