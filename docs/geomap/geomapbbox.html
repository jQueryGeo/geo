bbox

type bbox
init 
$( selector ).geomap( {
  bbox: function( e, geo ) { }
} );
bind 
$( selector ).bind( "geomapbbox", function( e, geo ) { } );
} );

The bbox event triggers any time user interaction causes a change in the current bbox of the map widget. This includes pan, marquee zoom, double-click to center/zoom, etc.

The geo argument is the new bbox.
