center

type Array (GeoJSON position)
default [ 0, 0 ]
init 
$( selector ).geomap( { center: [ 0, 0 ] } );
get 
var center = $( selector ).geomap( "option", "center" );
set 
$( selector ).geomap( "option", "center", [ -71.037598, 42.363281 ] );

The center property gets or sets the center point of the map. By default the value is in longitude/latitude degrees. You can change the default when you initialize the widget.
